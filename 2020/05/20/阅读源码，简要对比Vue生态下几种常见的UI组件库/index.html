From Computer Geek to Cult Icon
阅读源码，简要对比Vue生态下几种常见的UI组件库
发布于：2020-05-20 10:13:25 访问：

Element

用户量最大、文档最丰富、测试案例最完善的组件库，也是项目目前使用的组件库。
每个组件基本由两个文件组成:main.vue index.js
main.vue 负责业务逻辑，index.js 负责导出组件。

ViewUI

之前的iView，现在改名了，作者开始做一些商业上的尝试，作为独立开发者，可以算人生赢家。
代码中增加了JSX的支持，一些处理更优雅。

Vuetify

全部用ts实现，还使用了mixin来精简代码结构，这一点和我们项目的思路一样，所以感情上很认同。
